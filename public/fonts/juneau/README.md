# Juneau в админке

**Роль:** акцентный шрифт для заголовков (`font-juneau` + обычно `font-medium`). Основной текст админки — **Inter**.

Demo-файлы; для продакшена — полная лицензия, `@font-face` в `src/index.css`.

| Файл | Вес (CSS) |
|------|-----------|
| Juneau-Medium-Demo.otf | 500 |
| Juneau-Bold-Demo.otf | 700 |
| Light / Semi-Light | при необходимости для мелкого текста в Juneau |
