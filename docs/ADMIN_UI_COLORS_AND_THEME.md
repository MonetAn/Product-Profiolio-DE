# Цвета и тема админки (быстрый справочник)

**Назначение:** один файл, по которому агент и разработчик понимают, откуда берутся цвета в `/admin*` и что **не** смешивать с оранжевым дашборда (Brands).

## Как включается тема

- Маршруты админки рендерятся через **`AdminRoute`**.
- На корневой элемент документа (`<html>`) вешается класс **`admin-engineering`**.
- В `src/index.css` селектор **`html.admin-engineering`** переопределяет shadcn-токены (`--primary`, `--background`, …). Публичный дашборд без этого класса остаётся на **своём** `:root` (оранжевый primary Brands).

## Три уровня цвета

| Роль | Что использовать | Запрещено в админке |
|------|------------------|---------------------|
| **Primary / CTA** | Токены `primary`, `ring`, активные табы, ссылки | Оранжевый Brands как главный акцент |
| **«Нужно внимание» / валидация / подсветка полей** | В TSX: **`blue-*`** (Tailwind). В чистом CSS под админкой: **`hsl(var(--admin-attention))`** и **`--admin-attention-foreground`** (заданы только внутри `html.admin-engineering`) | **`amber-*`**, **`orange-*`**, жёлто-оранжевые hex «как на дашборде» — **кроме** явной семантической шкалы ниже |
| **Опасное действие** | `--destructive` / `destructive` | Смешивать с «вниманием» без смысла |

Переменные **`--admin-attention`** (~голубой DE, ~`#428EFF`) живут **только** в блоке `html.admin-engineering`; глобальный `:root` их не задаёт — оранжевый дашборда в админку не «просачивается» через эти токены.

### Исключение: шаг «Выберите кварталы для заполнения»

Блок **`AdminQuickFlowQuarterEffortOverview`** (шаг валидации Quick Flow) использует **отдельную** шкалу заполнения: **зелёный** (всё ок) → **янтарь** (план/факт метрик) → **красный** (блокеры карточки) → **серый** (заглушка). Это намеренно не DE-primary и не `--admin-attention`; не перекрашивать этот шаг в голубой без отдельного решения.

## Где уже завязано на токены

- **`src/styles/gantt.css`** — классы `*-admin-issue`, попап предупреждений: селекторы с префиксом `html.admin-engineering`, цвета через `--admin-attention`.
- **`src/styles/treemap.css`** — `.admin-effort-changed`: то же.
- Компоненты в **`src/components/admin/**`** — для подсветок предпочтительно **`blue-*`**, согласованно с `--admin-attention`.
