# Ревью текстов: эмоционально, «ты», супер лаконично

Правило: **«Пиши — сокращай»**. Всё коротко, неформально, с характером, обращение на «ты». Ниже — только то, что предлагается поменять; в конце блок «Оставляем как есть».

---

## 6. Другие экраны (кратко)

- **AdminAccess** (пустой список): «Пока никого нет. Добавьте первого пользователя выше.» → **«Тут пока никого. Добавь первого выше»** (ты, короче).
- **PeopleAssignmentsTable**: «Нет людей в выбранной команде» → **«В команде никого»**; «Нет инициатив» → **«Инициатив нет»**.
- **TreemapContainer / Gantt** empty state: заголовки типа «Нет инициатив по выбранным фильтрам» → **«Ничего не подходит»** или **«Подкрути фильтры»** (коротко).
- **InitiativeTable** (пусто): «Нет инициатив для отображения» → **«Здесь пусто»**.
- **AdminQuickFlow**: «Нет инициатив. Добавьте первую кнопкой ниже.» → **«Ни одной — добавь кнопкой ниже»**.

Индекс, дашборд, фильтры, диалоги (типы, описания, ссылки) — в этом ревью не меняем, только то, что перечислено выше. При необходимости сделаем второй проход.

## Оставляем как есть (без изменений)

- «Дашборд» / «Назад» (кнопка назад).
- «Заполнение», «Люди», «Доступ», «Активность» (табы).
- «N инициатив», «N из M», «N чел.» (статистика).
- **«Сохранено»** (бейдж статуса).
- Меню шестерёнки (все пункты).
- Все destructive-тосты и технические сообщения об ошибках.
- Placeholders в QuarterCell, InitiativeDetailDialog (числа, метрики, комментарии), если не указано иначе выше.
- Aria-labels и служебные подписи (оставляем понятными для доступности).
